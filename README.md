# Password-Reset

<h2>Description</h2>
This SOP will demenstrate how to change a User's Password in Active Directory
<br/>

<h2>Utilities Used</h2
                                 
- <b>Microsoft Azure</b>
- <b>Remmina (RDP client)</b>
- <b>Windows Server 2025</b>
- <b>Windows 11 Pro


<h2>Key Steps:</h2>

**1. Open Users and Computers in Server Manager**

- <b> In server manager select tools and open the Users and Computers menu</b>
- <b> Select the User folder under the domain to list all users</b>
- <b> Right click on the user and select password reset</b>

**2. Change Password**

- <b> Type in a new password for user</b>
- <b> Click okay to save changes</b>
- <b> Windows will notify you and confirm the password reset</b>

**3. Have User Login With New Password**

- <b> Open RDP client and change the password to the new password</b>
- <b> Save and connect to login</b>
- <b> Verify user can login with new password</b>
